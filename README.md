# Mathemagix - IIT-JEE (Mains+Advanced) Hub 🎓

A full-stack web application built for the Mathemagix educational institute. It features a modern frontend for students to explore courses and book demos, powered by a custom Python backend that handles email notifications and lead storage via MongoDB.

## 🚀 Tech Stack

**Frontend:**
- HTML5, CSS3, JavaScript (Vanilla)
- Fully responsive modern UI

**Backend:**
- Python 3.x
- Flask (Web Framework)
- PyMongo (MongoDB Integration)
- smtplib (Automated Email Notifications)
- Gunicorn (Production server)

## 📁 Project Structure
- `/frontend` - Contains all static HTML/CSS/JS files for the website.
- `/backend` - Contains the Flask API (`app.py`) and database logic.
